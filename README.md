# Marina Barbershop Website

Premium mobile-first website for **Marina Barbershop** in Journal Square, Jersey City, NJ.

## Business Info

- **Name:** Marina Barbershop
- **Address:** 865 Bergen Ave #2, Jersey City, NJ 07306
- **Phone:** (201) 736-3239
- **Walk-ins:** Always welcome

## Tech Stack

- [Next.js 16](https://nextjs.org/) with App Router
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- System font stack (`font-sans`)

## Pages

| Route | Description |
|---|---|
| `/` | Homepage — Hero, Services preview, Gallery, Hours, Testimonials, CTA |
| `/services` | Full services & pricing menu |
| `/gallery` | Masonry photo gallery |
| `/reviews` | Customer testimonials |
| `/contact` | Contact form, hours, and map |

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
npm start
```

## Design

Dark modern aesthetic with amber/gold accents (`amber-400`) on a zinc-950 background. Fully responsive and mobile-first.
